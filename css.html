  • css规范
  • 所有功能均有css注释
  • css样式均使用小写字母
  • 使用空格代替tab
  • 代码缩紧使用2个空格
  • 小程序项目均使用class
  • vue中局部的样式要设置socped
  • 样式命名不要过长，超过2级尽量使用简写。如果功能特殊可以写在最外层
  • 提取公共css到common.less文件中
  • 所有元素均要设置高度、如果展示内容不确定几行可以设置一个最小高度
  • 代码格式化
  • /* 样式书写一般有两种：一种是紧凑格式 (Compact) */
.jdc{ display: block;width: 50px;}

/* 一种是展开格式（Expanded）*/
.jdc{
    display: block;
    width: 50px;
}
  • 选择器
  • /* 尽量少用通用选择器 */
/* 不使用 ID 选择器 */
/* 不使用无具体语义定义的标签选择器 */

/* 推荐 */
.bxcc {}
.bxcc li {}
.bxcc li p{}

/* 不推荐 */
*{}
#bxcc {}
.bxcc div{}
  • 分号
  • /* 每个属性声明末尾都要加分号； */

.bxcc {
    width: 100%;
    height: 100%;
}
  • 代码易读性
  • /* 左括号与类名之间一个空格，冒号与属性值之间一个空格 */
/* 推荐：*/
.bxcc { 
  width: 100%; 
}

/* 不推荐：*/
.bxcc{ 
  width:100%;
}


/* 逗号分隔的取值，逗号之后一个空格 */
/* 推荐：*/
.bxcc {
  box-shadow: 1px 1px 1px #333, 2px 2px 2px #ccc;
}

/* 不推荐：*/
.bxcc {
  box-shadow: 1px 1px 1px #333,2px 2px 2px #ccc;
}


/* 为单个css选择器或新申明开启新行 */
/* 推荐：*/
.bxcc, 
.bxcc_logo, 
.bxcc_hd {
  color: #ff0;
}
.nav{
  color: #fff;
}

/* 不推荐：*/
.bxcc,bxcc_logo,.bxcc_hd {
  color: #ff0;
}.nav{
  color: #fff;
}


/* 颜色值 rgb() rgba() hsl() hsla() rect() */
/* 推荐：*/
.bxcc {
  color: rgba(255, 255, 255, 0.5);
}

/* 不推荐：*/
.bxcc {
  color: rgba( 255, 255, 255, .5 );
}


/* 不要为 0 指明单位 */
/* 推荐：*/
.bxcc {
  margin: 0 10px;
}

/* 不推荐：*/
.bxcc {
  margin: 0px 10px;
}
  • 属性值引号
  • /* css属性值需要用到引号时，统一使用单引号 */

/* 推荐 */
.jdc { 
	font-family: 'Hiragino Sans GB';
}

/* 不推荐 */
.jdc { 
	font-family: "Hiragino Sans GB";
}
  • 属性书写顺序
  • 建议遵循以下顺序:
  • 1、布局定位属性：display / position / float / clear / visibility / overflow
  • 2、自身属性：margin / padding / border / width / height / background
  • 3、文本属性：color / font / text-decoration / text-align / vertical-align / white- space / break-word
  • 4、其他属性（CSS3）：content / cursor / border-radius / box-shadow / text-shadow / background:linear-gradient …
  • .bxcc {
  display: block;
  position: relative;
  float: left;
  margin: 0 10px;
  padding: 20px 0;
  border: 1px;
  width: 100px;
  height: 100px;
  font-family: Arial, 'Helvetica Neue', Helvetica, sans-serif;
  color: #333;
  -webkit-border-radius: 10px;
  -moz-border-radius: 10px;
  -o-border-radius: 10px;
  -ms-border-radius: 10px;
  border-radius: 10px;
  background: rgba(0,0,0,.5);
}

  • CSS3浏览器私有前缀写法
  • /* CSS3 浏览器私有前缀在前，标准前缀在后 */

.bxcc {
  -webkit-border-radius: 10px;
  -moz-border-radius: 10px;
  -o-border-radius: 10px;
  -ms-border-radius: 10px;
  border-radius: 10px;
}
